## NPDS ROLLINPIM

A custom rollinpin build from scratch use it as a weapon in fivem

## Install

- ensure npds_plunger in youre server.cf
- Add the weapon to ox_inventory **Ox Inventory -> data -> weapons.lua** 

```
['WEAPON_ROLLINPIN'] = {
	label = 'Rollinpin',
	weight = 2500,
	durability = 0.1,
},
```
